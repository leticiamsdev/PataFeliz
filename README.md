# Pata Feliz 

**Veja a página no ar:** [https://leticiamsdev.github.io/PataFeliz/](https://leticiamsdev.github.io/PataFeliz/)

O **Pata Feliz** é uma landing page para um petshop especializado em serviços de Banho e Tosa. O site foi desenvolvido para ser moderno, responsivo e apresentar os serviços de forma clara e atrativa para os clientes.

## Descrição

O projeto consiste em uma landing page (página única) que detalha as etapas do processo de banho e tosa, desde a recepção até o toque final. A página inclui:
- **Início**: Apresentação do petshop e mensagem de boas-vindas.
- **Banho e Tosa**: Demonstração passo a passo (6 etapas) do cuidado que os pets recebem.
- **Avaliações**: Seção de depoimentos de clientes satisfeitos.
- **Responsividade**: Navegação adaptada para dispositivos móveis com um menu "hambúrguer".

## Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização da página, incluindo layout responsivo e design moderno.
- **JavaScript**: Lógica para o menu mobile e interatividade.
- **Bibliotecas Externas**:
  - [jQuery](https://jquery.com/): Para facilitar a manipulação do DOM.
  - [ScrollReveal](https://scrollrevealjs.org/): Para animações de rolagem (revelação de elementos ao descer a página).
  - [Font Awesome](https://fontawesome.com/): Para os ícones utilizados no projeto.

## Como Executar o Projeto

Como este é um projeto estático (front-end básico), não é necessário instalar dependências complexas ou configurar um servidor back-end.

1. Clone o repositório ou faça o download dos arquivos.
2. Extraia os arquivos, se necessário.
3. Dê um duplo clique no arquivo `index.html` para abri--lo em seu navegador de preferência (Google Chrome, Firefox, Edge, Safari, etc.).
   - *Alternativamente:* Você pode usar a extensão "Live Server" no VS Code para rodar o projeto com recarregamento automático.
